
HOS 汉字系统(小型珍藏版)
------------------------
                                                     简晶 于 96,12


一、 安装
---------
    HOS 可以安装到任何驱动器的任何目录或子目录中，系统目录名称可以随便
自己取。最后只要在 AUTOEXEC.BAT 中加入该目录路径，今后即可随意启动/退出
汉字系统。

二、基础使用
------------
    1. 启动汉字系统             HOS←┘
    2. 退出汉字系统             Q←┘ 或  CTRL+F5
    3. 系统设置                 CTRL+F10
    4. 提示行切换               右SHIFT(默认)
    5. 默认各种常用汉字输入
       ALT+F1           双拼双音
       ALT+F2           全拼
       ALT+F3           自然双拼
       ALT+F4           五笔
       ALT+F5           五笔划
       CTRL+ALT+F1      区位
       CTRL+ALT+F10     ASCII
       CTRL+ALT+L       联想开关(用ALT+F1输入法的词库进行联想)
       CTRL+ALT+G       屏幕取词(适用任何输入法)
       CTRL+ALT+J       自动记词开关(最后多敲一个空格确认,取消可按回车键)
       CTRL+空格        中/西文输入切换
       SHIFT+空格       全/半角输入切换(或 CTRL+F9)
       CTRL+ALT+空格    重复最后一次输入的字词
    6. 所有程序均可加上参数 "/?" 列出使用说明，请参考。
    7. HOS 仅提供一套16点阵和一套24点阵的宋体简体字库，同时供显示及打印用。
       只要安装 SuperVGA 显示卡，就可执行 24.COM 显示高清晰度的文字屏幕。

三、文件列表
------------
基本系统:
        HOS.BAT         汉字系统启动批处理
        Q.COM           汉字系统退出程序
        DDRV.EXE        安装显示驱动(命令行指定驱动程序名称, * 表示自动)
        TESTADP.EXE     显示卡即插即用程序(自动检测国内流行30余种显示卡)
        KRNL.EXE        系统核心模块
        GMODE.EXE       BIOS/VESA 模式设置程序
        16.COM          切换屏幕为16点阵字体
        24.COM          切换屏幕为24点阵字体
汉字输入:
        INT16.EXE       键盘管理模块(可直接支持WPS的任意输入法)
        IMD.EXE         万能输入法(可管理10张码表,命令行指定编码文件名称)
        ASE.EXE         中文全屏幕编辑器

四、目录结构
------------
    在汉字系统安装结束后，系统目录下面将有２个固定目录:

    <DRV>       存放各种驱动程序
    <DAT>       存放显示/打印字库及其他系统数据文件
